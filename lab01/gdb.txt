1. Аргументы к вашей программе могут быть заданы как аргументы к команде run. run arg1, arg2, ... или r <имя файла>. run без аргументов использует те же аргументы, которые использовались предыдущей командой run, или которые установлены командой set args.
2. С помощью команды b(break) <нормер строки>/<номер функциии>
3. С помощью команды n(next)
4. С помощью командыа s(step)
5. С помощью команды c(continue)
6. С помощью команды p(print) <имя переменной, выражение>
7. С помощью команды watch <имя переменной>
8. С помощью команды info locals
9. С помощью команды q(quit) или сочетанием клавиш ctrl+d
